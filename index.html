git reset HEAD

  • cofa zmiany, wracając do plików wyjściowych
    można regulować ile zrobionych przez nas commitów ma być cofniętych np git reset HEAD^ , cofnie nam o jeden, natomiast git reset HEAD~2 o dwa
  • gdy po modyfikowaniu, usuwaniu i dodawaniu wierszy w plikach, okazuje się, że zmiany nie są dobre i chcemy wrócić do wyjściowego pliku
  • git reset HEAD index.html

git remote add "remote_name" "git url"

komenda służąca do dodania nowego zdalnego repozytorium, która przyjmuje dwa argumenty: "remote_name" czyli nazwę zdalnego repozytorium oraz url do niego "git url". (Przykład: git remote add "nazwa" "https://github.com/user/repo.git")
